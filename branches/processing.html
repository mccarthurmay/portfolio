Theory

This piece demonstrates the usage of random movement and barriers.

Rectangle.js

  • Sets random speed for x and y position of rectangle.
  • Sets size of rectangle to 10x10 pixels.
  • The speed varies by + or - 2 pixels.

Sketch.js

  • Imports rectangle.js's rules for rectangle movement.
  • Sets boundaries for where the rectangle can move by reversing direction each time a certain plane is hit.
  • Creates a new rectangle at mouse position when left clicked.

Sierpinski Triangle

This piece shows multiple stages of the fractal design, Sierpinski Triangle, created by Wacław Franciszek Sierpiński.

  • Vectors are placed at three locations: a, b, and c.
  • X increases by 1 every time the draw function loops.
  • Each time x increases, a midpoint is created at half of the previous midpoints.
  • This continues until x = 6 or if the canvas is double left clicked.

Waves

This was my final project for a half-block Colorado College class called "Strange Fractals".

Setup

  • Sliders are created for the speed of circle size increase as well as maximum circle size.
  • Buttons for circle placement presets are created.
  • Textual instructions are also placed to help with user experience.

Draw

  • Sliders are defined.
  • The limit is set in two ways; the maximum size of the circle is user based, however the limit may be reached randomly. This is done to create variation.
  • RGB is also randomly given to the outline of each circle.

Controls

  • A button called "add" was created to allow random additions of circles within a given area.
  • The scroll wheel places a circle at mouse position.
  • Pressing escape pauses the fractal while pressing tab resumes the fractal.
  • A reset button was created to be able to reset the drawing without resetting the entire program.
  • 5 presets were created to test accurate placements of circles, which created varying designs.
